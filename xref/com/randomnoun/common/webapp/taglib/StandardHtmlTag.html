11  /**
12   * This is an abstract taglib, which can be used to propagate standard HTML attributes
13   * through to a HTML page. It provides handlers for all the 'general' attributes
14   * that are present for most HTML tags (e.g. onclick, style, ...).
15   *
16   * <p>Other taglibs that are used to generate HTML elements (e.g. SelectTag), can
17   * extend this class to take advantage of this functionality.
18   *
19   * <p>Superclasses may override any method defined in this tag to provide their own
20   * special handling.
21   *
22   * <b>The attributes handled by this tag</b> are
23        accesskey, styleClass, contenteditable, dir, disabled, lang, language, maxlength,
24        size, style, title, width, onactivate, onafterupdate, onbeforeactvate,
25        onbeforecut, onbeforedeactivate, onbeforeeditfocus, onbeforepaste,
26        onbeforeupdate, onblur, onchange, onclick, oncontextmenu, oncontrolselect,
27        oncut, ondblclick, ondeactivate, ondrag, ondragend, ondragenter, ondragleave,
28        ondragover, ondragstart, ondrop, onerrorupdate, onfilterchange, onfocus,
29        onfocusin, onfocusout, onhelp, onkeydown, onkeypress, onkeyup, onlosecapture,
30        onmousedown, onmouseenter, onmouseleave, onmousemove, onmouseout, onmouseover,
31        onmouseup, onmousewheel, onmove, onmoveend, onmovestart, onpaste,
32        onpropertychange, onreadystatechange, onresize, onresizeend, onresizestart,
33        onselect, onselectstart, ontimeerror, hidefocus, readonly, tabindex and
34        unselectable.
35   *
36   * <p>Note that the HTML attribute 'class' is now called 'styleClass', since the
37   * Object.getClass() method interferes with standard JavaBean reflection.
38   *
39   * <p>All attributes are lower-case, except extraAttributes (see below).
40   *
41   * <p>Note that the attributes 'name', 'type' and 'value' are intentionally not on this list,
42   * as they would normally be generated by the specific tag libary itself.
43   * 
44   * <p>The 'disabled' attributes has some special handling -- if it's set to the value 
45   * "false", then the attribute is removed. This is due to IE treating 'disabled="false"' as
46   * actually disabling the control. Which is odd.
47   *
48   * <p>The IE-only attributes hidefocus, readonly, tabindex and unselectable are included
49   * on this list, because they seem reasonably useful. The IE-only attributes
50   * begin, end, datafld, datasrc, autocomplete, syncmasterm, timecontainer and vcard_name
51   * are not on this list, because they don't seem as useful. You can always add them if
52   * you think they're needed.
53   *
54   * <p>Values are EL-evaluated (so you can have JSTL-style <tt>${xxxx}</tt> values).
55   *
56   * <p>Additional attributes can be expressed using the extraAttributes attribute, e.g.
57   * to put a non-standard attribute 'onmousewheelclick' into the element, you could
58   * use, for example:
59   * <pre style="code">
60   * &lt;mm:something extraAttribute="onmousewheelclick='doSomethingInJavascript()'" />
61   * </pre>

353     /** Returns a string with contains all attributes assigned to this tag, in key="value"
354      * format.
355      * 
356      * <p>(NB: values are not currently quote-escaped here) 
357      *
358      * @return All attributes assigned to this tag.
359      */
360     protected String getAttributeString() {
361         StringBuffer sb = new StringBuffer();
362         for (Iterator<Map.Entry<String, String>> i = attributes.entrySet().iterator(); i.hasNext();) {
363             Map.Entry<String, String> entry = i.next();
364             sb.append(entry.getKey() + "=\"" + entry.getValue() + "\" ");
365         }
366         return sb.toString() + (extraAttributes == null ? "" : extraAttributes);
367     }
