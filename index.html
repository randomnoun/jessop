randomnoun common-public

Common utility classes

Usage

To use this Maven project, include it in your project model 

<project>
  ...
  <dependencies>
    ...
    <dependency>
      <groupId>com.randomnoun.common</groupId>
      <artifactId>common-public</artifactId>
      <version>1.0.1</version>
    </dependency>
    ...    
  </dependencies>
  ...
</project>
